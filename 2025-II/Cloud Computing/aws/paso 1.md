
## 1️⃣ Configurar credenciales AWS (lo que antes te daba AWS Academy)

Antes AWS Academy te “regalaba” las credenciales temporalmente.  
Ahora **tú debes crearlas**.

### Paso A: Crear usuario IAM

En la consola AWS:

1. Ve a **IAM**
    
2. **Users → Create user**
    
3. Nombre: `serverless-user`
    
4. Marca **Access key – Programmatic access**
    
5. Permisos:
    
    - `AdministratorAccess` (para aprender, luego se ajusta)
        
6. Guarda:
    
    - **Access Key ID**
        
    - **Secret Access Key**
